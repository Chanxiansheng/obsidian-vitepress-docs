EasyFishingMod.readme ​

🎣 EasyFishing - 轻松钓鱼 MOD（DolocTown） ​

「减少机械操作，享受悠闲钓鱼时光。」

功能概览 ​

功能项说明
✅ 抛竿完美自动将抛竿距离锁定为“完美（最远）”，无需手动判断时机。
✅ 快速上钩缩短等待鱼上钩的时间，大幅减少等待过程。
✅ 自动上钩鱼上钩后自动收竿，无需点击确认。
✅ 跳过小游戏直接跳过钓鱼小游戏，一键获得结果。
🎚️ 钓鱼额外耗时（分钟）成功钓鱼后自动跳过指定时间，用于平衡收益或减少日程安排影响。
🎯 自动钓鱼（快捷键）一键启动挂机钓鱼：启动后将循环钓鱼行为，直到你手动按任意键中断。
（实际上循环使用鱼竿，偷懒不想搞状态机了，会强制启用配置项以确保走完钓鱼流程）

安装依赖 ​

  • 需安装（前置）
    • BepInEx
  • 建议安装（见上图，设置会注入游戏 UI 面板中）
    • ModSettingManagerForDolocTown.dll
  • 可选安装（需自己放入插件文件夹中）
    • ConfigurationManager
