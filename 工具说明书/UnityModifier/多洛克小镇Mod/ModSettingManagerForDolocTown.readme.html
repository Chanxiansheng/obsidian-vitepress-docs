ModSettingManagerForDolocTown.readme ​

ModSettingManagerForDolocTown 是一款为《多洛可小镇》《DolocTown》设计的基于 BepInEx 的轻量级设置注入插件，支持 mod 开发者将配置项集成至游戏原生设置面板中。

功能特性 ​

  • 支持的设置类型：Toggle、Slider、Dropdown、KeyBindDropdown（按键绑定）

  • 基于 BepInEx.ConfigEntry：配置项独立存储，支持 ConfigurationManager 可视化兼容，不参与游戏本身的设置存储

  • **支持多个 mod 共用，支持 Section 分组

  • 提供友好的 API 调用方式 (Fluent 风格)

  • 不保证没 BUG

一些碎碎语 ​

  • 很喜欢这个游戏的风格，难得国内有这么精致的种田游戏，游戏创意工坊上线似乎还有一段时间，暂时先凑合着用，希望能撑到开放对应接口。

  • 本来只是想简单写一个简单的钓鱼 Patch Mod，减轻我的腱鞘炎，写着写着就想试一下注入面板，又写着写着就抽象成一个独立插件，现在好啦，腱鞘炎更严重了。

  • 游戏中的按键系统实在是太硬编码了，没有那个能力做组件级复刻，只能用 Dropdown 了

  • 目前只实现了单向数据流，即游戏 UI -> ConfigEntry -> ConfigurationManager，由于游戏 UI 面板是行为驱动的，非数据驱动，强行实现双向数据响应很困难，因此现在 ConfigurationManager 中的行为没有办法让 UI 面板实时响应。
