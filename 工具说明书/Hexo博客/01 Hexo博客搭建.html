https://hexo.io/zh-cn/docs/
https://butterfly.js.org/posts/21cfbf15/

一、Hexo简介 ​

Hexo 是一个基于Nodejs的快速、简洁且高效的博客框架。
特点：全静态页面，可以和github pages服务搭配，进而无需服务器。社区资源较全。
推荐主题：butterfly、Next...

二、搭建步骤 ​

0补充：修改npm库源

powershell
// 修改npm源
npm config set registry  https://registry.npm.taobao.org/

  1. 安装Hexo(局部)

    powershell
    $ npm install hexo

  2. 新建项目

    powershell
    $ npx hexo init <folder>
$ cd <folder>
$ npm install

  3. 配置项目配置

    • 修改_config.yml配置文件，见hexo文档。

  4. 安装自动部署工具

    powershell
    $ npm install hexo-deployer-git --save

  5. 修改git托管配置（_config.yml）

    yaml
    deploy:
  type: git
  repo: <repository url>
  branch: [branch]
  message: [message]

  6. 部署

    powershell
    $ npx hexo clean 
$ npx hexo deploy

三、优化设置 ​

  1. 配置个人域名

    1. 配置域名映射
    2. GitHub Pages设置中把个人域名添加到Custom domain
    3. source中创建一个名为CNAME文件，不要后缀，写上域名

  2. 主题安装

    • 通过 npm 安装并不会在 themes 里生成主题文件夹，而是在 node_modules 里生成
    powershell
    npm i hexo-theme-butterfly
