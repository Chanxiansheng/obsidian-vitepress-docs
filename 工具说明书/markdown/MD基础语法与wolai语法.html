MD 基础语法与 wolai 语法 ​

一、标题 ​

使用 # 号可表示 1-6 级标题，一级标题对应一个 # 号，二级标题对应两个 # 号，以此类推。#号和文字间至少得有一个空格

md
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

wolai 中同样可以使用 # 标题 ，也可以使用 /h1~h6

二、字体 ​

  1. 文本强调
    使用星号（*）和底线（_）包裹文字进行标记。一个表示斜体，两个表示粗体，三个表示粗斜体。

    md
    *斜体文本*  
*斜体文本*

**粗体文本**  
**粗体文本**

***粗斜体文本***  
___ 粗斜体文本 ___

  2. 删除线
    使用两个波浪线（~~）包裹

    md
    ~~删除线~~

    wolai 中可以使用单个波浪线包裹 ~标题~

三、引用 ​

在引用的文字前加>即可，引用也可以嵌套。

markdown
> 这是引用的内容
>> 这是引用的内容
>>> 这是引用的内容

wolai 中可以使用 > 内容 (中间需要加空格)，也可以使用 /yswz

四、分割线 ​

三个或者三个以上的（-）或者 （*） 都可以。

markdown
---
-----
***
*****

wolai 中可以使用 ---，也可以使用 /fgx

五、图片 ​

markdown
![图片alt](图片地址)

图片alt就是显示在图片下面的文字，相当于对图片内容的解释。
图片title是图片的标题，当鼠标移到图片上时显示的内容。

wolai 中可以使用 /tp

六、超链接 ​

markdown
[超链接名](超链接地址)

实例
[百度](http://baidu.com)

wolai 中可以使用 /wzsq 或 /bookmark

七、列表 ​

无序列表 ​

无序列表用 - + * 任何一种都可以

markdown
- 列表内容
+ 列表内容
* 列表内容

注意：- + * 跟内容之间都要有一个空格

wolai 中可以使用 + 内容 (中间需要加空格)，也可以使用 /wxlb

有序列表 ​

数字 + 加点 + 空格

markdown
1. 列表内容
2. 列表内容
3. 列表内容

注意：序号跟内容之间要有空格

wolai 中可以使用 1. 内容 (中间需要加空格)，也可以使用 /szlb

列表嵌套 ​

上一级和下一级之间敲三个空格即可

八、表格 ​

markdown
表头|表头|表头
---|:--:|---:
内容|内容|内容
内容|内容|内容

第二行分割表头和内容。
- 有一个就行，为了对齐，多加了几个
:--- 文字默认居左
:--: 表示文字居中
---: 表示文字居右

注：原生的语法两边都要用 | 包起来。此处省略

实例：
姓名|技能|排行
--|:--:|--:
刘备|哭|大哥
关羽|打|二哥
张飞|骂|三弟

wolai 中可以直接插入简单表格或数据表格

九、代码 ​

单行代码：代码之间分别用一个反引号包起来

markdown
 `代码内容`

代码块：代码之间分别用三个反引号包起来，且两边的反引号单独占一行

markdown
```js
  代码...
  代码...
  代码...
```

wolai 同理
