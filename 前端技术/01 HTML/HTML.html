简介 ​

什么是 HTML？ ​

  • HTML 全称Hyper Text Markup Language ，是一种超文本标记语言。
  • 标记语言是一套标记标签。

HTML 标签 ​

  • HTML 标签是由尖括号包围的关键词，如 <html>。
  • HTML 标签也可称为 HTML元素。
  • HTML 通常为双标签<a></a>，也有单标签<hr>。
  • HTML 标签间的关系分为包含关系和并列关系。
html
<!-- 包含关系 -->
<head>
	<title>Document</title>
</head>

<!-- 并列关系 -->
<head>
</head>
<body>
</body>

HTML 结构 ​

一个初始化的 HTML 结构如下图所示

  • <!DOCTYPE>
    • <!DOCTYPE> 声明是文档类型声明（Document Type Declaration）的缩写，用于指示浏览器当前页面使用的 HTML 版本。
    • <!DOCTYPE html> 声明按照 HTML5 规范进行渲染。
  • <html> 元素是整个 HTML 页面的根元素。
  • <head>
    <head> 是所有头部元素的容器，包含了文档的标题，也可以包含脚本、样式、meta 信息以及其他更多的信息。
    • <meta charset="utf-8"> 定义了网页编码格式为 utf-8。
    • <title> 元素描述了文档的标题。
  • <body> 元素包含了文档所有的可见的页面内容。

HTML 属性 ​

  • 属性是 HTML 元素提供的附加信息。
  • 属性一般描述于开始标签
  • 属性总是以名称/值对的形式出现，比如：name="value"。
  • 不同 HTML 元素所能定义的属性是不同的。

常用的 HTML 属性：

属性描述
class为 html 元素定义一个或多个类名（classname）(类名从样式文件引入)
id定义元素的唯一 id
style规定元素的CSS 行内样式（inline style）

头部 <head> ​

<head> 元素包含了所有的头部标签元素。在 <head>元素中你可以插入脚本（scripts）, 样式文件（CSS），及各种 meta 信息。

可以添加在头部区域的元素标签为: <title>, <style>, <meta>, <link>, <script>, <noscript> 和 <base>。

<title> （必需） ​

<title> 定义了不同文档的标题，在 HTML 文档中是必需的。

<title>Document</title>

<meta> ​

<meta> 标签描述了一些基本的元数据。

<meta> 标签提供了元数据。元数据也不显示在页面上，但会被浏览器解析。

<meta> 元素通常用于指定网页的描述、关键词、文件的最后修改时间、作者和其他元数据。

元数据可以使用于浏览器（如何显示内容或重新加载页面），搜索引擎（关键词），或其他 Web 服务。

html
<!-- 为搜索引擎定义关键词: -->
<meta name="keywords" content="HTML, CSS, XML, XHTML, JavaScript">

<!-- 为网页定义描述内容: -->
<meta name="description" content="免费 Web & 编程 教程">

<!-- 定义网页作者: -->
<meta name="author" content="Runoob">

<!-- 每30秒钟刷新当前页面: -->
<meta http-equiv="refresh" content="30">

<!-- 定义视窗大小，常用于 媒体查询  -->
<meta name="viewport" content="width=device-width">

<base> ​

<base> 标签描述了基本的链接地址/链接目标，该标签作为 HTML 文档中所有的链接标签的默认链接。

<base href="http://xxxx.com" target="_blank">

<link> ​

<link> 标签定义了文档与外部资源之间的关系，常用于引入外部的 CSS 样式表。

<link rel="stylesheet" type="text/css" href="mystyle.css">

<style> ​

<style> 标签定义了 HTML 文档的样式文件引用地址，称为内部 CSS 样式表。

html
<style type="text/css">
  body {
    background-color:yellow;
  }
  p {
    color:blue
  }
</style>

<script> ​

<script> 标签用于加载脚本文件，如：JavaScript。

<script> 元素既可包含脚本语句，也可以通过 "src" 属性引入外部脚本文件。

html
<!-- 脚本语句 -->
<script>
  document.write("Hello World!")
</script>

<!-- 引入外部脚本文件 -->
<script src="[url]"></script>

<noscript> ​

<noscript> 元素用来定义在脚本未被执行时的替代内容（文本）。

此标签可被用于可识别 <noscript> 标签但无法支持其中的脚本的浏览器。

html
<script>
document.write("Hello World!")
</script>
<noscript>抱歉，你的浏览器不支持 JavaScript!</noscript>

基础元素 ​

元素区块分类 ​

可视的 HTML 元素可以被分为块级元素和行内元素。

  • 块级元素（display:block）

块级元素独占一行，可以设置宽度、高度以及边距等样式属性。它适合用于创建页面的大块结构，例如页面的主体区域、容器、布局等。
例如: <h1>, <p>, <ul>, <table>,<div>

  • 行内元素（display:inline）

内联元素不会独占一行，宽度默认由其内容决定。它适合用于对文本或其他行内元素进行样式化、标记或包裹。
例如: <a>、<strong>、<b>、<em>、<i>、<span>

  • 行内块元素（display:inline-block）

行内块元素同时具有块元素的特点和行内元素的特点。

  • 和相邻的行内元素（包含行内块）在一行上，它们直接会有空白缝隙。
  • 一行可以显示多个（行内元素特点）
  • 默认宽度就是内容的宽度（行内元素特点）
  • 高度、宽度、内外边距都可以自定义（块元素特点）
    例如: <img />、<input />、<td>

块<div> ​

<div> 元素是块级元素，没有特定的含义，它可用于组合其他 HTML 元素的容器。

如果与 CSS 一同使用，<div> 元素可用于对大的内容块设置样式属性。

行内块<span> ​

<span> 元素是行内元素，也没有特定的含义，可用作文本的容器。

当与 CSS 一同使用时，<span> 元素可用于为部分文本设置样式属性。

标题<h1>-<h6> ​

<h1> - <h6> 标签可定义标题。块级元素

总共有六个层级的标题，<h1> 定义最大的标题。 <h6> 定义最小的标题。

html
<h1>一级标题</h1>
<h2>二级标题</h2>
<h3>三级标题</h3>
<h4>四级标题</h4>
<h5>五级标题</h5>
<h6>六级标题</h6>

段落<p> ​

HTML 可以将文档分割为若干段落，通过 <p> 标签定义。块级元素

<p>这是一个段落 </p>

链接<a> ​

HTML 使用标签 <a> 来设置超文本链接。行内元素

<a>标签可以包裹一段文字、一个图片或者其他的元素，通过点击这些内容来跳转到新的文档或者当前文档中的某个部分。

html
<a href="url" target="_blank" title="这是链接的title">链接文本</a>

<a href="url"><img src="url"></img></a>

  • 常用属性

    • href（必选）：指定链接目标的 URL，这是链接的最重要属性。可以是另一个网页的 URL、文件的 URL 或其他资源的 URL。
    • target（可选）：指定链接如何在浏览器中打开。常见的值包括 _blank（在新标签或窗口中打开链接）和 _self（在当前标签或窗口中打开链接）。
    • title（可选）：提供链接的额外信息，通常在鼠标悬停在链接上时显示为工具提示。
    • rel（可选）：指定与链接目标的关系，如 nofollow、noopener 等。

  • 默认样式（伪类相关）

    • 鼠标悬停默认样式为小手。（cursor:pointer)
    • 字体带有下划线（text-decoration: underline）
    • 一个未访问过的链接显示为蓝色字体并带有下划线。(:link{color:blue})
    • 访问过的链接显示为紫色并带有下划线。(:visited{color:purple})
    • 点击链接时，链接显示为红色并带有下划线。(:active{color:red})

  • 锚点
    除了链接到其他网页外，您还可以在同一页面内创建内部链接，这称为锚点链接。

    • 创建锚点
      id 属性可用于创建一个 HTML 锚点，也可以使用**<a>的 name 属性定义**。
    • 指向锚点
      <a>标签的 href 属性使用#符号指向该锚点。
    html
    <footer id="footer ">页脚</footer >

 <a href="#footer">锚点</a>  
 <a href="xxxx.com#footer">外链锚点</a>

📌 注意：前端路由的hash 模式会使锚点功能失效

图像<img> ​

在 HTML 中，图像由<img> 标签定义。

<img> 是空标签，意思是说，它只包含属性，并且没有闭合标签。

html
<img src="url" alt="some_text" loading="lazy">
  • 常用属性
    • src（必选）：src 指 "source"，用于规定显示图像的 URL。
    • alt（必选）：规定图像的替代文本，在浏览器无法载入图像时，替换文本属性告诉读者她们失去的信息。
    • height/width（可选）：用于设置图像的高度与宽度，单位像素。
    • loading（可选）：指定浏览器是应立即加载图像 eager 还是延迟加载图像 lazy。

框架<iframe> ​

当需要在同一个浏览器窗口中显示不止一个页面，或者嵌入其他外部页面，可以使用 <iframe> 标签。行内块元素

html
<iframe src="URL"></iframe>
  • 常用属性
    • src（必选）：src 指 "source"，用于规定插入页面的 URL。
    • height/width（可选）：用于设置插入页面的高度与宽度，单位像素。

当网页不希望被嵌套, 可以配置响应头 X-Frame-Options，可选值：
DENY：表示该网站页面不允许被嵌套，即便是在自己的域名的页面中也不能进行嵌套。
SAMEORIGIN：表示该页面可以在相同域名页面中被嵌套展示。
ALLOW-FROM uri：表示该页面可以在指定来源页面中进行嵌套展示。

辅助元素 ​

换行<br> ​

当希望在不产生一个新段落的情况下进行换行（新行），可以使用 <br> 标签。

水平线<hr> ​

<hr> 标签定义 HTML 页面中的主题变化（比如话题的转移），并显示为一条水平线。

<hr> 元素被用来分隔 HTML 页面中的内容（或者定义一个变化）。

注释<!-- --> ​

<!--...--> 注释标签用来在源文档中插入注释。注释不会在浏览器中显示。

html

字符实体 ​

HTML 中的预留字符必须被替换为字符实体。

  • 浏览器总是会截短 HTML 页面中的空格。如果您在文本中写 10 个空格，在显示该页面之前，浏览器会删除它们中的 9 个。如需在页面中增加空格的数量，您需要使用 &nbsp; 字符实体。
  • 在 HTML 中不能使用小于号（<）和大于号（>），这是因为浏览器会误认为它们是标签。如果希望正确地显示预留字符，我们必须在 HTML 源代码中使用字符实体。如需显示小于号，我们必须这样写成 &lt 或 &#60。
显示结果描述实体名称实体编号
空格&nbsp;&#160;
<小于号&lt;&#60;
>大于号&gt;&#62;
&和号&amp;&#38;
"引号&quot;&#34;

列表元素 ​

HTML 支持有序、无序和定义列表，列表项内部可以使用段落、换行符、图片、链接以及其他列表等等。行内元素

无序列表<ul><li> ​

无序列表是一个项目的列表，此列项目使用粗体圆点（典型的小黑圆圈）进行标记。

无序列表始于 <ul> 标签。每个列表项始于 <li>。

html
<ul>
  <li>item1</li>
  <li>item2</li>
</ul>

有序列表<ol><li> ​

有序列表也是一列项目，列表项目使用数字进行标记。

有序列表始于 <ol> 标签。每个列表项始于 <li> 标签。

html
<ol>
  <li>item1</li>
  <li>item2</li>
</ol>

定义列表<dl><dt><dd> ​

自定义列表不仅仅是一列项目，而是项目及其注释的组合。

自定义列表以 <dl> 标签开始。 <dt>用于定义项名 。 <dd>定义描述。

html
<dl>
  <dt>HTML</dt>
  <dd>制作网页的标准语言，控制网页的结构</dd>
  <dt>CSS</dt>
  <dd>层叠样式表，控制网页的样式</dd>
  <dt>JavaScript</dt>
  <dd>脚本语言，控制网页的行为</dd>
</dl>

表格元素 ​

表格由 <table> 标签来定义。块级元素

<table> 元素表示整个表格，它包含两个主要部分：<thead> 和 <tbody>。

  • <thead > 用于定义表格的标题部分: 在 <thead> 中，使用 <tr> 元素定义标题行，使用 <th> 元素定义列的标题。
  • <tbody > 用于定义表格的主体部分: 在 <tbody > 中，使用 <tr> 元素定义行，并在每行中使用 <td > 元素定义单元格数据。

每个表格均有若干行（由 <tr> 标签定义），每行被分割为若干单元格（由 <td> 标签定义）。表格可以包含标题行（<th>）用于定义列的标题。

  • th：th 是 table header 的缩写，表示表格的表头单元格。
  • tr：tr 是 table row 的缩写，表示表格的一行。
  • td：td 是 table data 的缩写，表示表格的数据单元格。

数据单元格可以包含文本、图片、列表、段落、表单、水平线、表格等等。

html
<!-- 表格示例 -->
<table border="1">
  <thead>
    <tr>
      <th>列标题1</th>
      <th>列标题2</th>
      <th>列标题3</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>行1，列1</td>
      <td>行1，列2</td>
      <td>行1，列3</td>
    </tr>
    <tr>
      <td>行2，列1</td>
      <td>行2，列2</td>
      <td>行2，列3</td>
    </tr>
  </tbody>
</table>

    https://www.w3school.com.cn/svg/index.asp

  • SVG 形状

    SVG 有一些预定义的形状元素，可被开发者使用和操作：

    • 矩形 <rect>
    • 圆形 <circle>
    • 椭圆 <ellipse>
    • 线 <line>
    • 折线 <polyline>
    • 多边形 <polygon>
    • 路径 <path>

  • SVG 优势

    与其他图像格式相比（比如 JPEG 和 GIF），使用 SVG 的优势在于：

    • SVG 图像可通过文本编辑器来创建和修改
    • SVG 图像可被搜索、索引、脚本化或压缩
    • SVG 是可伸缩的
    • SVG 图像可在任何的分辨率下被高质量地打印
    • SVG 可在图像质量不下降的情况下被放大

  • Canvas 与 SVG 的比较

CanvasSVG

HTML5 新特性 ​

拖放 API ​

拖放是一种常见的特性，即抓取对象以后拖到另一个位置。

在 HTML5 中，拖放是标准的一部分，任何元素都能够拖放。

设置元素为可拖放，需要把 draggable 属性设置为 true 。

<img draggable="true">

每一个可拖动的元素，在拖动过程中，都会经历三个过程，拖动开始-->拖动过程中--> 拖动结束。

针对对象事件名称说明
被拖动的元素dragstart在元素开始被拖动时候触发
drag在元素被拖动时反复触发
dragend在拖动操作完成时触发
目的地对象dragenter当被拖动元素进入目的地元素所占据的屏幕空间时触发
dragover当被拖动元素在目的地元素内时触发
dragleave当被拖动元素没有放下就离开目的地元素时触发

dragenter 和 dragover 事件的默认行为是拒绝接受任何被拖放的元素。因此，我们必须阻止浏览器这种默认行为。e.preventDefault()
