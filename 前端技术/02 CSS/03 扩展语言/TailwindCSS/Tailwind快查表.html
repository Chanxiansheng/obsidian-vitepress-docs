伪类参考表 ​

  • :hover ​

当用户将鼠标光标悬停在元素上时，使用 hover 变体设置元素的样式：

html
<div class="bg-black hover:bg-white ...">  <!-- ... --></div>

  • :focus ​

使用 focus 变体对元素获得聚焦时进行样式化：

html
<input class="border-gray-300 focus:border-blue-400 ..." />

  • :focus-within ​

使用 focus-within 变体对元素或其后代之一获得聚焦时进行样式化：

html
<div class="focus-within:shadow-lg ...">  <input type="text" /></div>

  • :focus-visible ​

当元素获得聚焦时使用使用 focus-visible 变体对键盘进行样式化：

html
<button class="focus-visible:outline-2 ...">Submit</button>

  • :active ​

使用 active 变体对元素被按下时进行样式化：

html
<button class="bg-blue-500 active:bg-blue-600 ...">Submit</button>

  • :visited ​

使用 visited 变体为已访问的链接设置样式：

html
<a href="https://seinfeldquotes.com" class="text-blue-600 visited:text-purple-600 ..."> Inspiration </a>

  • :target ​

如果元素的 ID 与当前 URL 片段匹配，则使用 target 变体为其设置样式：

html
<div id="about" class="target:shadow-lg ...">  <!-- ... --></div>

  • :first-child ​

如果元素是其类型的第一个子元素，则使用 first 变体为其设置样式：

html
<ul>  
	{#each people as person}    
	<li class="py-4 first:pt-0 ...">     
	<!-- ... -->    
	</li> 
	{/each}
</ul>

  • :last-child ​

如果元素是其类型的最后一个子元素，则使用 last 变体为其设置样式：

html
<ul>
  {#each people as person}
  <li class="py-4 last:pb-0 ..."> <!-- ... --> </li>
  {/each}
</ul>

  • :only-child ​

如果元素是唯一子元素，则使用 only 变体为其设置样式：

html
<ul>
  {#each people as person}
  <li class="py-4 only:py-0 ..."> <!-- ... --> </li>
  {/each}
</ul>

  • :nth-child(odd) ​

如果元素是奇数编号子元素，则使用 odd 变体为其设置样式：

html
<table>
  {#each people as person}
  <tr class="bg-white odd:bg-gray-100 ..."> <!-- ... --></tr>
  {/each}
</table>

  • :nth-child(even) ​

如果元素包含内容，则设置元素样式它是一个使用 even 变体的偶数编号子元素：

html
<table>
  {#each people as person}
  <tr class="bg-white even:bg-gray-100 ..."> <!-- ... --> </tr>
  {/each}
</table>

  • :first-of-type ​

如果元素是其类型的第一个子元素，则使用 first-of-type 变体为其设置样式：

html
<nav>
  <img src="/logo.svg" alt="Vandelay Industries" />
  {#each links as link}
  <a href="#" class="ml-2 first-of-type:ml-6 ..."> <!-- ... --> </a>
  {/each}
</nav>

  • :last-of-type ​

如果元素是其类型的最后一个子元素，则使用 last-of-type 变体为其设置样式：

html
<nav>
  <img src="/logo.svg" alt="Vandelay Industries" />
  {#each links as link}
  <a href="#" class="mr-2 last-of-type:mr-6 ..."> <!-- ... --> </a>
  {/each}
  <button>More</button>
</nav>

  • :only-of-type ​

如果元素是其类型的唯一子元素，则使用 only-of-type 变体为其设置样式：

html
<nav>
  <img src="/logo.svg" alt="Vandelay Industries" />
  {#each links as link}
  <a href="#" class="mx-2 only-of-type:mx-6 ..."> <!-- ... --> </a>
  {/each}
  <button>More</button>
</nav>

  • :nth-child() ​

使用 nth 变体为特定位置的元素设置样式：

html
<nav>
  <img src="/logo.svg" alt="Vandelay Industries" />
  {#each links as link}
  <a href="#" class="mx-2 nth-3:mx-6 nth-[3n+1]:mx-7 ..."> <!-- ... --> </a>
  {/each}
  <button>More</button>
</nav>

  • :nth-last-child() ​

使用 nth-last 变体为从末尾开始的特定位置的元素设置样式：

html
<nav>
  <img src="/logo.svg" alt="Vandelay Industries" />
  {#each links as link}
  <a href="#" class="mx-2 nth-last-3:mx-6 nth-last-[3n+1]:mx-7 ..."> <!-- ... --> </a>
  {/each}
  <button>More</button>
</nav>

  • :nth-of-type() ​

使用 nth-of-type 变体为特定位置的相同类型的元素设置样式：

html
<nav>
  <img src="/logo.svg" alt="Vandelay Industries" />
  {#each links as link}
  <a href="#" class="mx-2 nth-of-type-3:mx-6 nth-of-type-[3n+1]:mx-7 ..."> <!-- ... --> </a>
  {/each}
  <button>More</button>
</nav>

  • :nth-last-of-type() ​

使用 nth-last-of-type 变体为从末尾开始的特定位置的相同类型的元素设置样式：

html
<nav>
  <img src="/logo.svg" alt="Vandelay Industries" />
  {#each links as link}
  <a href="#" class="mx-2 nth-last-of-type-3:mx-6 nth-last-of-type-[3n+1]:mx-7 ..."> <!-- ... --> </a>
  {/each}
  <button>More</button>
</nav>

  • :empty ​

使用 empty 变体为没有内容的元素设置样式：

html
<ul>
  {#each people as person}
  <li class="empty:hidden ...">{person.hobby}</li>
  {/each}
</ul>

  • :disabled ​

当输入被禁用时，使用 disabled 变体设置输入的样式：

html
<input class="disabled:opacity-75 ..." />

  • :enabled ​

当输入被启用时，使用 enabled 变体设置输入的样式，当你只想在元素未被禁用时应用另一种样式时，这种方法最有用：

html
<input class="enabled:hover:border-gray-400 disabled:opacity-75 ..." />

  • :checked ​

使用 checked 变体为选中的复选框或单选按钮设置样式：

html
<input type="checkbox" class="appearance-none checked:bg-blue-500 ..." />

  • :indeterminate ​

使用 indeterminate 变体为处于不确定状态的复选框或单选按钮设置样式：

html
<input type="checkbox" class="appearance-none indeterminate:bg-gray-300 ..." />

  • :default ​

使用 default 变体设置页面初始加载时的默认值选项、复选框或单选按钮样式：

html
<input type="checkbox" class="default:outline-2 ..." />

  • :optional ​

当输入为可选时，使用 optional 变体设置输入的样式：

html
<input class="border optional:border-red-500 ..." />

  • :required ​

当输入为必需时，使用 required 变体设置输入的样式：

html
<input required class="border required:border-red-500 ..." />

  • :valid ​

当输入有效时，使用 valid 变体设置输入的样式：

html
<input required class="border valid:border-green-500 ..." />

  • :invalid ​

当输入无效时，使用 invalid 变体设置输入的样式：

html
<input required class="border invalid:border-red-500 ..." />

  • :user-valid ​

当输入有效且用户与其交互时，使用 user-valid 变体设置其样式：

html
<input required class="border user-valid:border-green-500" />

  • :user-invalid ​

当输入无效且用户与其交互时，使用 user-invalid 变体设置其样式：

html

  • :in-range ​

<input min="1" max="5" class="in-range:border-green-500 ..." />

video,
canvas,
audio,
iframe,
embed,
object {
  display: block; /* 1 */
  vertical-align: middle; /* 2 */
}

/*
  Constrain images and videos to the parent width and preserve their intrinsic aspect ratio. (https://github.com/mozdevs/cssremedy/issues/14)
*/

img,
video {
  max-width: 100%;
  height: auto;
}

/*
  1. Inherit font styles in all browsers.
  2. Remove border radius in all browsers.
  3. Remove background color in all browsers.
  4. Ensure consistent opacity for disabled states in all browsers.
*/

button,
input,
select,
optgroup,
textarea,
::file-selector-button {
  font: inherit; /* 1 */
  font-feature-settings: inherit; /* 1 */
  font-variation-settings: inherit; /* 1 */
  letter-spacing: inherit; /* 1 */
  color: inherit; /* 1 */
  border-radius: 0; /* 2 */
  background-color: transparent; /* 3 */
  opacity: 1; /* 4 */
}

/*
  Restore default font weight.
*/

:where(select:is([multiple], [size])) optgroup {
  font-weight: bolder;
}

/*
  Restore indentation.
*/

:where(select:is([multiple], [size])) optgroup option {
  padding-inline-start: 20px;
}

/*
  Restore space after button.
*/

::file-selector-button {
  margin-inline-end: 4px;
}

/*
  Reset the default placeholder opacity in Firefox. (https://github.com/tailwindlabs/tailwindcss/issues/3300)
*/

::placeholder {
  opacity: 1;
}

/*
  Set the default placeholder color to a semi-transparent version of the current text color in browsers that do not
  crash when using `color-mix(…)` with `currentcolor`. (https://github.com/tailwindlabs/tailwindcss/issues/17194)
*/

@supports (not (-webkit-appearance: -apple-pay-button)) /* Not Safari */ or
  (contain-intrinsic-size: 1px) /* Safari 17+ */ {
  ::placeholder {
    color: color-mix(in oklab, currentcolor 50%, transparent);
  }
}

/*
  Prevent resizing textareas horizontally by default.
*/

textarea {
  resize: vertical;
}

/*
  Remove the inner padding in Chrome and Safari on macOS.
*/

::-webkit-search-decoration {
  -webkit-appearance: none;
}

/*
  1. Ensure date/time inputs have the same height when empty in iOS Safari.
  2. Ensure text alignment can be changed on date/time inputs in iOS Safari.
*/

::-webkit-date-and-time-value {
  min-height: 1lh; /* 1 */
  text-align: inherit; /* 2 */
}

/*
  Prevent height from changing on date/time inputs in macOS Safari when the input is set to `display: block`.
*/

::-webkit-datetime-edit {
  display: inline-flex;
}

/*
  Remove excess padding from pseudo-elements in date/time inputs to ensure consistent height across browsers.
*/

::-webkit-datetime-edit-fields-wrapper {
  padding: 0;
}

::-webkit-datetime-edit,
::-webkit-datetime-edit-year-field,
::-webkit-datetime-edit-month-field,
::-webkit-datetime-edit-day-field,
::-webkit-datetime-edit-hour-field,
::-webkit-datetime-edit-minute-field,
::-webkit-datetime-edit-second-field,
::-webkit-datetime-edit-millisecond-field,
::-webkit-datetime-edit-meridiem-field {
  padding-block: 0;
}

/*
  Remove the additional `:invalid` styles in Firefox. (https://github.com/mozilla/gecko-dev/blob/2f9eacd9d3d995c937b4251a5557d95d494c9be1/layout/style/res/forms.css#L728-L737)
*/

:-moz-ui-invalid {
  box-shadow: none;
}

/*
  Correct the inability to style the border radius in iOS Safari.
*/

button,
input:where([type='button'], [type='reset'], [type='submit']),
::file-selector-button {
  appearance: button;
}

/*
  Correct the cursor style of increment and decrement buttons in Safari.
*/

::-webkit-inner-spin-button,
::-webkit-outer-spin-button {
  height: auto;
}

/*
  Make elements with the HTML hidden attribute stay hidden by default.
*/

[hidden]:where(:not([hidden='until-found'])) {
  display: none !important;
}
